Key Software Used in EP1000

Software Used Reasons For Using Software
Fusion360 by Autodesk Used for CAD drawings, mainly to create 3D designs in preparation for 3D printing and laser cutting in this module. The software is flexible and easy to navigate, making it a common choice for 3D designing.
Inkscape A free software that is used for editing vectors. It was used to prepare and edit vectors before carrying out laser cutting.
Sublime Text Used to edit text for web development. Its features make web development more manageable.
Arduino IDE Used to edit, verify and utilize codes for embedded systems.
CorelDRAW Used to finalise the vector and engraving designs before sending it to be printed with the laser cutting software.
GitHub Desktop Allows changes to be updated to this website.
